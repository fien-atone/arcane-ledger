# TTRPG Companion — Functional Requirements Document

> **Статус:** черновик v0.2 — 20.03.2026
> Документ живой — пополняется по ходу проектирования.
> Сортировка внутри разделов: 🔴 M → 🟠 S → 🟡 C → ⚪ W

---

## Приоритеты (MoSCoW)

| Метка | Значение |
|-------|----------|
| 🔴 **M** — Must have | Без этого продукт не работает |
| 🟠 **S** — Should have | Важно, но можно выпустить без |
| 🟡 **C** — Could have | Хорошо иметь, если время позволяет |
| ⚪ **W** — Won't have | Сознательно исключено из скоупа |

## Категории

| Метка | Значение |
|-------|----------|
| `core` | Фундаментальная механика платформы |
| `gm` | Функциональность для Мастера |
| `player` | Функциональность для Игрока |
| `sharing` | Механики раскрытия и совместного доступа |
| `meta` | Управление аккаунтом, кампаниями, настройки |
