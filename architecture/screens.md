# TTRPG Companion — Screens & Blocks

> **Статус:** черновик v0.1 — 20.03.2026
> Для каждого экрана: назначение, блоки, кто видит блок.
> Видимость: 🧙 GM only | ⚔️ Player only | 👥 Both (но контент может отличаться)

---

## Навигационная структура

```
/ — лендинг (публичный)
└── /login — вход

/app
├── /campaigns                        — список всех кампаний пользователя
└── /campaigns/:id
    ├── / (dashboard)                 — главная кампании
    ├── /npcs                         — список NPC
    ├── /npcs/:id                     — карточка NPC
    ├── /party                        — состав партии
    ├── /characters/:id               — карточка PC
    ├── /sessions                     — список сессий
    ├── /sessions/:id                 — страница сессии
    ├── /locations                    — список локаций
    ├── /locations/:id                — карточка локации
    ├── /factions                     — список фракций
    ├── /factions/:id                 — карточка фракции
    ├── /quests                       — список квестов
    ├── /quests/:id                   — карточка квеста
    └── /materials                    — библиотека материалов

/admin                                — только для admin-пользователя
├── /users                            — управление пользователями
└── /users/:id                        — редактирование пользователя
```

### 2. Campaign Dashboard `/campaigns/:id` ✅
**Назначение:** Главная страница кампании. Отвечает на вопрос "где мы сейчас и что дальше?". GM — командный центр, игрок — окно в мир через своего персонажа.

| Блок | GM 🧙 | Player ⚔️ | Заметки |
|------|--------|-----------|---------|
| **Заголовок кампании** — название, обложка, статус (активна / завершена) | ✅ | ✅ | Статус "завершена" меняет вид дашборда — архивный режим |
| **Глобальный поиск** по кампании | ✅ | ✅ | Ищет по NPC, локациям, сессиям, квестам |
| **Последняя сессия** — дата, краткое содержание, незакрытые хвосты (`nextSessionNotes`) | ✅ | ✅ | Ключевой блок. Ссылка → страница сессии |
| **История сессий** — ссылка-переход на полный список с датами | ✅ | ✅ | Не разворачивается на дашборде — только переход |
| **Дата следующей игры** | ✅ | ✅ | Простое поле на кампании; будущее — интеграция с календарём |
| **Активные квесты** — краткий список (название, статус) + переход на все квесты | ✅ все | ✅ только открытые | Будущая фича: избранные / закреплённые квесты |
| **Мой персонаж** — фото, имя, вид (species), класс | — | ✅ | Краткая карточка; инвентарь и детали — будущее |
| **Состав группы** — карточки PC: фото, имя, вид, класс + имя игрока | ✅ | ✅ | Переход на карточку персонажа |
| **Быстрые переходы** — реестры NPC / Локаций / Фракций | ✅ | ✅ | Иконки-ярлыки, не разворачиваются |
| **Быстрые действия** — добавить NPC / сессию / локацию / квест | ✅ | — | |
| **Управление участниками** | ✅ | — | |

> Q-09 закрыт: "Мой персонаж" — закреплённая карточка на дашборде, детали через сайдбар → Персонажи.

### 5. Party `/campaigns/:id/party` ✅
**Назначение:** Состав партии — все игроки кампании и их персонажи.

| Блок | GM 🧙 | Player ⚔️ |
|------|--------|-----------|
| Карточки PC (фото, имя, вид, класс, игрок) | ✅ | ✅ |
| Игрок без персонажа — визуальный placeholder "нет персонажа" | ✅ | ✅ |
| gmNotes по каждому персонажу | ✅ | — |
| Кнопка "Добавить персонажа" | ✅ | — |

> При создании персонажа GM сразу выбирает игрока из списка участников кампании — без игрока персонаж не создаётся.
> Игрок приглашён, но персонажа нет → карточка-заглушка с именем игрока и статусом "персонаж не создан".
